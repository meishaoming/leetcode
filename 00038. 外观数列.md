[38. 外观数列](https://leetcode-cn.com/problems/count-and-say/)

Look-and-say Sequence

## 1

```py
class Solution:
    def countAndSay(self, n: int) -> str:
        s = "1"
        for _ in range(1, n):
            x = ''
            num = 1
            repeat = s[0]
            for ch in s[1:]:
                if repeat == ch:
                    num += 1
                else:
                    x += str(num) + repeat
                    num = 1
                    repeat = ch
            else:
                s = x + str(num) + repeat
        return s
```

## 2

一个小改动是在 s[1:] 后面加一个空格字符，以简化边界的处理。

```py
class Solution:
    def countAndSay(self, n: int) -> str:
        s = "1"
        for _ in range(1, n):
            x = ''
            num = 1
            repeat = s[0]
            for ch in s[1:]+" ":
                if repeat == ch:
                    num += 1
                else:
                    x += str(num) + repeat
                    num = 1
                    repeat = ch
            s = x
        return s
```
