[141. 环形链表](https://leetcode-cn.com/problems/linked-list-cycle/)

快慢指针。如果是首尾相连形成的最大环，则需要花费 n 步才能追上。

* 时间复杂度 O(n)
* 空间复杂度 O(1)。

```py
class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        fast = slow = head
        while fast and fast.next:
            fast = fast.next.next
            slow = slow.next
            if fast == slow:
                return True
        return False
```
