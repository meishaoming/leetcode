[66. 加一](https://leetcode-cn.com/problems/plus-one/)

## 解法一

```py
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        carry = False
        for i in range(len(digits)-1, -1, -1):
            digits[i] += 1
            if digits[i] >= 10:
                digits[i] -= 10
                carry = True
            else:
                carry = False
                break
        if carry:
            digits.insert(0, 1)
        return digits
```

写简单点

```py
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits)-1, -1, -1):
            digits[i] += 1
            if digits[i] < 10:
                return digits
            else:
                digits[i] -= 10
        return [1] + digits
```
