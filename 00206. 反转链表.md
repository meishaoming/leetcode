## 1 迭代

```py
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        prev, cur = None, head
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        return prev
```

* 时间复杂度 O(n)。n 为链表上的结点个数
* 空间得杂度 O(1)

## 2 递归

递归的解法理解起来反倒是比较绕

```
    head           tail
     |              |
     v              v
1 -> 2 -> 3 <- 4 <- 5
```

每次递归把上一次的 head 保存在了栈里边。递归回溯的时候又取出来。

`head.next = None` 是为了在 head 指到最头部的时候把它的 next 指向 None 。

```py
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        if not head or not head.next:
            return head
        tail = self.reverseList(head.next)
        head.next.next = head
        head.next = None
        return tail
```
